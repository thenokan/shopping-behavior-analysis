# shopping-behavior-analysis
Анализ поведения покупателей на основе данных о покупках
DataLens: [https://datalens.ru/rdd8ku42bd5ub-proekt](https://datalens.yandex/rdd8ku42bd5ub)
Проект представляет собой комплексный анализ данных о покупательском поведении, основанный на датасете из 1375 записей о покупках. Анализ выполнен с использованием Python и включает полный цикл работы с данными: от загрузки и предобработки до визуализации и формулирования бизнес-рекомендаций.
